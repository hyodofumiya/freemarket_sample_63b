フリマアプリ

既存のフリマアプリを参考にしたクローンサイトです。

概要

このアプリはフリーマーケットアプリケーションです。

既存のフリマサービスのサイトを参考に、学習のためにチームで制作したクローンサイトです。

商品の出品・閲覧・編集・購入機能を備えています。

特徴

  1. カテゴリー機能を実装し、商品検索をスムーズに

    • 大カテゴリー・中カテゴリー・小カテゴリーを実装することで、見やすさと検索しやすさを備えたカテゴリー機能を実装

  2. フリマアプリとしての必要最低限の機能を実装

    • 商品出品を初め、マイページやお気に入り機能、コメント機能、クレジットカード決済機能などを実装

担当領域

開発環境

  • OS : MacOS
  • 言語 : Ruby version 2.5.1
  • FW : Ruby on Rails version 5.2.4
  • DB : MySQL
  • サーバー : AWS(EC2,S3)
