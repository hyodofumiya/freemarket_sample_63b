フリマアプリ

チームで制作したフリマアプリの個人ポートフォリオです。

概要

学習の一貫で制作したフリーマーケットのアプリケーションです。

スクールの最終課題にて作成したアプリケーションを紹介します。

特徴

  1. 既存サービスと同様の機能を実装

    • 商品関連はもちろん、マイページやお気に入り機能、コメント機能、クレジットカード決済機能などを実装しています。

  2. 詳細なカテゴリー機能を実装し、商品検索をスムーズに

    • 大カテゴリー・中カテゴリー・小カテゴリーを実装することで、検索のしやすさや見やすさ向上につなげています。

担当領域

  • フロントエンド
    • トップページ
    • 商品詳細ページ
    • 商品購入ページ
  • サーバーサイド
    • カテゴリー機能
    • 商品購入機能
  • スクラムマスター
    • タスク管理
    • 進捗管理

開発環境

  • OS : MacOS
  • 言語 : Ruby version 2.5.1
  • FW : Ruby on Rails version 5.2.4
  • DB : MySQL
  • サーバー : AWS(EC2,S3)
  • 開発手法：アジャイル

機能一覧

商品関連
